# uni-vue3-template

## 说明

**该项目是`Uniapp` vue3 快速开发模板,vscode 版本基于`vite`构建，内置了`tailwindcss`以及部分业务场景的封装**

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn h5
```

### Compiles and minifies for production

```
yarn build:h5
```
